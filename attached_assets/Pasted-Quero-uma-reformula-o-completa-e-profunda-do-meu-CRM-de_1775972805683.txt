4. PALETA DE CORES
============================================================

Usar como base principal a linguagem cromática do MODELO 1.

Definir um sistema de cor claro, consistente e reutilizável.

BASE:
- Background principal: cinza muito claro / gelo
- Surface / cards: branco
- Border: cinza muito suave
- Texto principal: cinza escuro quase preto
- Texto secundário: cinza médio
- Texto fraco: cinza claro
- Primária: azul forte e limpo semelhante ao MODELO 1
- Positivo: verde moderno e suave
- Negativo: vermelho ou rosa suave
- Atenção: laranja suave
- Destaque secundário: roxo somente em elementos pontuais de gráfico ou badge, sem competir com o azul

Regras:
- azul deve ser a cor principal de ação
- usar verde apenas para sucesso/crescimento/conclusão
- vermelho/rosa para perda/queda/alerta
- laranja para atenção
- roxo apenas como apoio sutil em visualizações secundárias
- não escurecer a interface
- manter predominância clara
