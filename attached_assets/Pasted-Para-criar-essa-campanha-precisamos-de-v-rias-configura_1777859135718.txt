Para criar essa campanha, precisamos de várias configurações específicas. No entanto, é importante notar que o regulamento apresentado é complexo e envolve várias condições que não podem ser abordadas por uma única campanha no sistema.

### Situações que Exigem Múltiplas Campanhas:

1. **Duas comissões/prêmios diferentes por categoria de vendedor**: A campanha tem duas categorias (Incentivo e Elite) com comissões diferentes (0,5% e 1%, respectivamente).
2. **Múltiplos rankings simultâneos**: A campanha requer dois rankings (Maior Vendas e Crescimento).
3. **Categoria Incentivo que pode "subir" para Elite**: Vendedores da Categoria Incentivo que ultrapassam R$ 60.000,00 devem ser tratados como Elite.

### Configuração Proposta:

Dada a complexidade, sugiro criar **quatro campanhas separadas** para abordar todas as condições:

1. **Campanha DTR Incentivo**: Para vendedores com metas individuais (Alan, Janio, Erivan, Mariane, Carlisson) com comissão de 0,5% sobre Não-Tubos.
2. **Campanha DTR Elite**: Para vendedores com meta mínima de R$ 60.000,00, com comissão de 1% sobre Não-Tubos.
3. **Ranking Maior Vendas**: Para premiar o vendedor com o maior faturamento bruto no trimestre, apenas para vendedores que atingiram R$ 60.000,00.
4. **Ranking Crescimento**: Para premiar o vendedor com o maior crescimento percentual em relação ao mesmo período do ano anterior, com regra especial para cálculo de crescimento.

### Notas Importantes:

- **Trava coletiva de crescimento da loja**: O sistema não calcula automaticamente a trava de crescimento da loja. É necessário verificar manualmente se a loja atingiu o crescimento de 25% antes de processar a apuração.
- **Mix percentual entre categorias**: O sistema não calcula o mix percentual entre categorias (Conexões/Tubos). É necessário verificar manualmente se o vendedor atingiu o mix mínimo de 40% de Conexões.
- **Base mínima para crescimento**: O sistema não considera uma base mínima para o cálculo de crescimento. Para o ranking de crescimento, é necessário aplicar manualmente a regra especial de usar R$ 60.000,00 como base mínima para vendedores sem histórico ou com vendas abaixo de R$ 60.000,00 no período anterior.
