3. PALETA DE CORES
================================================================

Usar o MODELO 1 como base principal de cor.

- cards: branco puro
- texto principal: cinza escuro quase preto
- texto secundário: cinza médio
- bordas: cinza claro bem sutil
- cor primária: azul vivo semelhante ao MODELO 1
- azul principal para ações, botões primários, links, estados ativos, ícones importantes e gráficos principais
- verde para crescimento, positivo, concluído, metas batidas

Direção:

- nada de dark theme como base
- nada de fundo preto na versão principal
